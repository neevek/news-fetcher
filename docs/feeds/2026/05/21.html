Claude Code v2.1.147：Pinned background sessions 与 `/code-review` 改版

后台会话与 `/code-review` 工作流大改。

Claude Code v2.1.147 重点调整后台会话生命周期、代码审查命令和自动更新体验，同时修复企业登录限制与终端输出问题。该版本还移除了旧 /simplify 的清理修复行为，转向更明确的 review 工作流。

  • Pinned background sessions（在 claude agents 中用 Ctrl+T 固定）空闲时保持存活，更新后原地重启，内存压力下优先保留到非 pinned sessions 之后。

  • /code-review 现在报告 correctness bugs，旧的 cleanup-and-fix 行为已移除。
  • Auto-updater 增强：重试瞬时网络失败，失败时报告错误类别、OS error code，并显示当前版本。
  • 大文件编辑的 diff rendering 性能改进。
  • Prompt history 不再记录连续重复输入。
  • 修复 forceLoginOrgUUID 和 forceLoginMethod managed settings 未约束 third-party-provider 与 API-key sessions 的问题。
  • 修复 ! command 输出中的 & 被显示为 &amp;，导致复制 gcloud auth login 等 URL 出错的问题。
